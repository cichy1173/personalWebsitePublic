How does this website work?

It is not just a simple website that runs somewhere on a server. Why? I am glad you are asking ;-).

This site uses Hugo Framework. Hugo FW is a static website generator and it is ideal for personal blog or landing page. It is light, simple and uses technologies that I really love, for example Markdown. Also it needs to be light because this website does not run on a cloud server. It runs on my own hardware.

This schema shows how this is working:

This website runs on my HP 800 G3 Mini computer that works as my server and/or self-hosted homelab. It also uses YunoHost environment and my_webapp_ynh application. Code is hosted on GitHub and is being cloned into Mini PC with simple command.

That’s how this page runs on my own computer with domain bought from Gandi.
